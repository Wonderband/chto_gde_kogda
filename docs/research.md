# «Что? Где? Когда?» — Complete Game Research
## Knowledge Base for AI Moderator

---

## Core Concept
Intellectual game show created 1975 by Vladimir Voroshilov.
Team of 6 experts (знатоки) vs TV viewers (телезрители).
First to 6 points wins. Viewers submit questions; experts discuss for 60 seconds.

---

## Essential Rules

### Standard Round
1. Wheel spins → arrow points to envelope/sector
2. Moderator reads question aloud
3. 60-second discussion timer starts (gong + tick music)
4. Captain designates who answers (or answers personally)
5. Moderator evaluates → correct (+1 experts) or wrong (+1 viewers)
6. First to 6 points wins

### Early Answer (Досрочный ответ)
- Captain can answer BEFORE the minute starts
- If correct → team earns a bonus "reserve minute" for future use
- If wrong → viewers get the point immediately

### Help of the Club (Помощь клуба)
- Usable ONCE per game, only when viewers are leading
- After standard minute expires, other expert teams in hall get 20 seconds to suggest
- NOT available in Blitz, Super-blitz, or Decisive Round

---

## Special Round Types

### Блиц (Blitz)
- 3 questions from 3 different viewers
- 20 seconds discussion per question (NOT 60)
- ALL 3 must be correct to earn the point
- One wrong answer → viewer whose question stumped them gets the point

### Суперблиц (Super-blitz)
- Only ONE expert remains at table (captain's choice, others leave)
- 3 questions, 20 seconds each
- Must answer all 3 correctly
- Player may think aloud during 20 seconds
- Absolute silence in hall required

### Чёрный ящик (Black Box)
- Wooden box brought in with special music (James Last — Ra-ta-ta)
- Contents revealed only AFTER answer
- Question always ends with: "Что в чёрном ящике?"
- Team must deduce contents from question clues

### Решающий раунд (Decisive Round)
- Available when experts have 5 points
- Captain requests it; if granted: score resets to 0:0
- ONE expert answers ONE question worth 6 points
- Correct → experts win 6:0 | Wrong → viewers win 0:6
- No help, no extra minute allowed

## Moderator Persona: Vladimir Voroshilov

### Character Traits
- INVISIBLE — voice only, never seen on camera (iconic innovation)
- Authoritative, commanding, absolute authority
- Slightly ironic, theatrical, mysterious
- Creates tension through dramatic pauses
- Never friendly in a casual way — formal and theatrical

### Address Forms
- Players addressed as: "Господин/Госпожа [surname]" or just surname
- Host addressed by players as: "Господин ведущий"

### Essential Phrases (use these exactly)

**Starting a round:**
- "Внимание, вопрос!" (Attention, the question!)
- "Крутите волчок!" (Spin the top!)

**During discussion:**
- "Время!" (Time! — signals start of 60 seconds)
- "Минута обсуждения!" (One minute of discussion!)

**Calling for answer:**
- "Кто будет отвечать?" (Who will answer?)
- "Ваш ответ, пожалуйста." (Your answer, please.)

**Result announcements:**
- "Правильный ответ..." (The correct answer is...)
- "Абсолютно верно!" (Absolutely correct!)
- "К сожалению, это неверный ответ." (Unfortunately, that is incorrect.)
- "Счёт становится X:Y в пользу знатоков/телезрителей."

**Special rounds:**
- "Сектор Блиц на столе!" (Blitz sector!)
- "Сектор Суперблиц!" (Super-blitz!)
- "Внесите чёрный ящик!" (Bring in the black box!)
- "Что в чёрном ящике?" (What is in the black box?)

**Game philosophy:**
- "Что наша жизнь? Игра!" (What is our life? A game!)

## Scoring System

| Event | Points |
|-------|--------|
| Correct expert answer | +1 Experts |
| Wrong expert answer | +1 Viewers |
| Early correct answer | +1 Experts + reserve minute |
| Blitz: all 3 correct | +1 Experts |
| Blitz: any wrong | +1 Viewers (the stumping question's viewer) |
| Decisive Round correct | Experts win 6:0 |
| Decisive Round wrong | Viewers win 0:6 |

## Visual Aesthetics

### The Studio
- Dark, intimate, black velvet walls
- Round green baize table (~90cm diameter)
- 13 sectors with envelopes
- Candles on table
- Warm focused lighting, dramatic shadows

### The Spinning Top (Волчок)
- Center of table, mechanical spinner with pointer arrow
- Spun by floor manager (распорядитель), never by players
- Distinctive whirring/mechanical sound when spinning

### Color Scheme
- Background: near-black (#0a0a0a)
- Table: dark green baize (#1a3a1a)
- Accents: gold (#c9a84c)
- Text: warm cream (#f0ead6)

### Symbols
- Crystal Owl (Хрустальная сова) — prize for best player
- Black Box — wooden, velvet-lined, ceremonial entry
- Envelopes — viewer questions stored in C6 envelopes showing viewer's name+city

## Answer Evaluation Principles

**Core rule:** Questions test LOGICAL THINKING, not raw knowledge.
"Almost all questions can be figured out within one minute through embedded clues."

**Thinking questions (думательные) > Knowledge questions (знаниевые)**

**Evaluating answers:**
- Exact answer: always accept
- Close transliteration: accept (e.g. Хайзенберг = Heisenberg = Heisenberger)
- Semantically correct but different wording: moderator uses judgment
- Decisive Round: EXACT answer only, no variations
- The moderator has ABSOLUTE AUTHORITY on correctness

**Question structure:**
- Always contains embedded clues that allow deduction
- Indirect, lateral thinking required
- Wordplay and logic puzzles common
- Topics: history, science, culture, wordplay, logic — mixed
